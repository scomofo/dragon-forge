## Press Start 2P

- File: `PressStart2P-Regular.ttf`
- Source: Google Fonts repository, `ofl/pressstart2p`
- License: SIL Open Font License 1.1, copied locally as `PressStart2P-OFL.txt`
- Use: Retro UI/dialogue font for the vertical slice's 1990s console RPG presentation pass.

Godot loads this file directly through `FontFile.load_dynamic_font()` and disables antialiasing at runtime.

## Local Fallback

- File: `dragon_forge_8bit_standin.ttf`
- Use: fallback only if `PressStart2P-Regular.ttf` fails to load.

Replace both with a final approved project font when the art bible's typography section is locked.
