# Victory State: The Patch

## Purpose

The Mirror Admin victory should not restore the world to its old pastoral state. Skye jams the Hard Reset and forces the simulation into a Stable Hybrid render mode.

The end state is not fantasy hiding hardware anymore. It is a new aesthetic where both truths coexist.

## Stable Hybrid Render

Visual direction:

- Hand-painted terrain remains warm and readable.
- Circuitry settles into bark, stone, water, and dragon scales.
- No constant flicker between 16-bit and 4K.
- Glitches become scars and affordances, not failures.

## Mirror Admin Resolution

The Mirror Admin is not deleted.

It is de-compiled into a non-hostile System Familiar:

- Follows Skye.
- Provides real-time world health data.
- Warns of unresolved tickets.
- Acts as order/caution to Skye's improvisation.

## Final Sequence

1. Mirror Admin Packet Integrity reaches zero or the Hard Reset threshold.
2. TransitionController begins CRT collapse.
3. If Skye has the Analog Manual / Manual Override relic, HUD prompts for override.
4. Player interrupts collapse.
5. CRT collapse pauses halfway.
6. Stable Hybrid patch applies.
7. Mirror Admin enters Read-Only / Familiar state.
8. Service Tickets become VERIFIED where complete.

## End-Credits Map

After stabilization:

- The map unfolds into the full Hardware Husk schematic.
- Completed Service Tickets are marked `VERIFIED`.
- Free-roam remains open.
- System Daemon dragon can fly to leftover Analog Relics and hidden 16-bit developer Easter Eggs.

## Performance Note

Post-processing should stay centralized:

- Prefer one transition CanvasLayer / SubViewport path for CRT, dither, and glitch effects.
- Avoid stacking many full-screen shader nodes during flight.

## Asset Structure Note

Use inherited dragon scenes:

- `Dragon.tscn` base.
- Inherited Magma, Shadow, System Daemon, and Administrator Avatar variants.
- Shared movement/combat code stays in the base scene.
